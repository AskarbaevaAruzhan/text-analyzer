
Жоба туралы
Бұл жоба мәтіндегі сөздерді талдап, әр сөздің қанша рет кездесетінін есептейді. Сонымен қатар жалпы сөз санын көрсетеді.

Мақсаты
- Мәтінді өңдеу (String)
- Сөздерді бөлу
- Сөз жиілігін анықтау
- HashMap қолдану арқылы деректер сақтау

 Қалай жұмыс істейді
1. Пайдаланушы мәтін енгізеді  
2. Мәтін кіші әріпке айналдырылады  
3. Тыныс белгілер жойылады  
4. Мәтін сөздерге бөлінеді  
5. Әр сөздің жиілігі есептеледі  
6. Нәтиже экранға шығарылады
 Қолданылған технологиялар
- Java
- HashMap
- String әдістері (split, replaceAll, toLowerCase)
- For цикл
